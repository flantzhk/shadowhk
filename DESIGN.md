# ShadowHK — Design System (distilled from ShadowHK-Design-Reference.md + Branding.html, canonical: src/styles/variables.css)

**Palette "Sage & Vermilion"** (light, warm paper): bg-0 #F4F1EC page cream; bg-1 #FAFAF7 cards; bg-2 #EDE7DF secondary; bg-3 #E4DDD5 borders. Text: fg-0 #1A1A1A, fg-1 #3D3D3D, fg-2 #6B6B6B, fg-3 #ABABAB. Accent vermilion #C8392B (single action color), blush tint #F2DDD9, glow rgba(200,57,43,.2). Immersive screens: near-black #0B1417 stage.

**Type:** Serif display (Source Serif 4) for headlines and editorial moments; DM Sans for UI; JetBrains Mono for eyebrows/jyutping labels. Eyebrow labels: mono, letterspaced, uppercase. Scale contrast ≥1.25 between steps. 12px floor.

**Layout:** Print/newspaper aesthetic: sharp 0-radius corners on hero images, section cards, continue boxes (deliberate); radius only on chips/pills/modals (8-24px). Editorial section bars (thick rules + numbered eyebrows "01 PICK BACK UP"). Spacing scale 4/8/12/16/20/24/32/40. Warm shadows only.

**Breakpoints:** mobile-first 428px column; ≥900px = desktop (sidebar 240px + content canvas).

**Motion:** 200ms ease-out standard; no bounce; respect reduced-motion.

**Buttons:** vermilion filled primary (pill), quiet text/outline secondary. One accent color carries all interaction.
